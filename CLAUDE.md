# קייטרינג אלגריה — הנחיות פרויקט

אתר תדמית + לידים ל**קייטרינג אלגריה** (קרית גת). React + Vite + Tailwind (SPA),
נפרס ב-Vercel ל-https://www.alegriacatering.co.il.

## מותג
- השם הוא **"קייטרינג אלגריה"** — לא "מסעדה". אל תשתמש ב"מסעדת/המסעדה".
- מיצוב: הקייטרינג/הסעדה הגדול בדרום. 25+ שנה, קרית גת.
- ארבעה שירותי דגל: הסעדה יומית למפעלים (הליבה), מכירת שישי, קייטרינג לאירועים,
  ארוחות מסובסדות לעובדים.

## כלל הקופי (חובה — על כל טקסט חדש שעולה לאתר)
כל קופי שיווקי בעברית עובר את שכבת **human-ad-copy** לפני שהוא עולה. בפרט:
- **בלי הקבלה שלילית** ("זה לא X, זה Y") ובלי "לא רק... אלא".
- **בלי שלשות** ("טעים, טרי ואיכותי"). פריט אחד חזק או אסימטריה.
- **בלי אוצר מילים שרוף**: מהפכני, פורץ דרך, חדשני, מושלם, איכותי (כתואר ריק), חוויה, מסע.
- **קו מפריד ארוך (—): מקסימום אחד לעמוד**, עדיף אפס. פסיק/נקודה/נקודתיים במקום.
- **קונקרטי מעל עמום**: קרית גת, 054-543-7526, הזמנות עד רביעי, 3,000 מנות ביום,
  עוף בזיתים / חריימה / מטבוחה — במקום "אלפי לקוחות מרוצים".
- **מבחן קריאה בקול**: אם אי אפשר להגיד את זה בטלפון בלי להישמע כמו ברושור — לשכתב.
- קצב אסימטרי, סימן קריאה אחד לכל היותר לנכס.

## ארכיטקטורה
- כל הקופי ב-`src/data/*.js` (Content-in-Data). `site.js` = מקור אמת יחיד.
- SEO: `scripts/postbuild-seo.mjs` מזריק head לכל route + מפרה-רנדר גוף מאמרים + sitemap/robots.
- מדידה: `src/lib/analytics.js` (Meta Pixel פעיל; GA4/Google Ads/GTM env-driven).
- לידים: מודאל 3 שלבים → וואטסאפ + פופ-אפ אישור.

## בלוג
- מאמרים = קבצי `.md/.mdx` ב-`content/blog/` (מערכת חיצונית דוחפת אותם ל-main).
- כותב ברירת מחדל: עמית בן שושן. `draft: true` = מוסתר.
