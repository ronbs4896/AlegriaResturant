# פונטים · Fonts

כאן מעלים את קובצי הפונט של המותג.

**הפונט הראשי:** `Fb IguanaNew`

## מה להעלות לכאן
- `Fb IguanaNew` בפורמט `.woff2` (מועדף), `.otf` או `.ttf`
- רצוי כמה משקלים אם יש: Regular / Medium / Bold / Black

> אם יש רק פורמט אחד (למשל `.ttf`) — גם מצוין, אני ממיר ל-`.woff2` לביצועים.

עד שהפונט יגיע, האתר משתמש ב-fallback עברי איכותי (Heebo / Assistant).
