# תמונות · Images

מעלים כל תמונה לתיקייה המתאימה. ככל שיש יותר תמונות איכותיות — האתר חי ומשכנע יותר.

| תיקייה | מה שמים כאן |
|---|---|
| `logo/` | לוגו אלגריה — גרסה בהירה + כהה (PNG שקוף / SVG) |
| `hero/` | תמונות רקע גדולות ורוחביות לראש העמוד (אווירה, שולחן ערוך, מטבח) |
| `dishes/` | מנות ואוכל — תקריבים מפתים |
| `catering/` | אירועי קייטרינג, בופה, אירועים עסקיים |
| `ambience/` | אווירה — המקום, המטבח, הצוות, מאחורי הקלעים |
| `gallery/` | תמונות נוספות לגלריה הכללית |

## טיפים
- פורמט: `.jpg` לתמונות · `.png`/`.svg` ללוגו
- שמות קבצים באנגלית ובלי רווחים (למשל `friday-meal-01.jpg`)
- רזולוציה גבוהה — אני דואג לאופטימיזציה בבנייה
